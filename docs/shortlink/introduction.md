## English

## What is ADIYH Shortlink?

Shortlink is a built-in feature of ADIYH that goes beyond simply shortening URLs. It acts as a powerful traffic manager with the ability to distribute traffic across multiple destination URLs based on percentage.

### Key Features:

- **Traffic Split / Distribution**: You can direct a single shortlink to multiple target URLs and control the traffic share for each target using percentages (e.g., 50% to URL A, 50% to URL B).
- **Referer Manipulation (Beta)**: An advanced feature allowing you to manipulate the HTTP referer data.

## Indonesian

## Apa itu ADIYH Shortlink?

Shortlink adalah fitur bawaan ADIYH yang tidak hanya berfungsi untuk memendekkan URL. Fitur ini bertindak sebagai pengelola traffic yang canggih dengan kemampuan membagi traffic ke beberapa URL tujuan berdasarkan persentase.

### Fitur Utama:

- **Pembagian Traffic (Traffic Split)**: Anda bisa mengarahkan satu shortlink ke beberapa target URL sekaligus dan mengatur porsi traffic untuk masing-masing target menggunakan persentase (contoh: 50% ke URL A, 50% ke URL B).
- **Manipulasi Referer (Beta)**: Fitur lanjutan yang memungkinkan manipulasi data HTTP referer.
