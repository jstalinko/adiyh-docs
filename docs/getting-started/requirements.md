# Requirements

To install and run **ADIYH Plugin** properly, make sure your server environment meets the following requirements:

## Server Requirements

- **WordPress**: Latest stable version
- **PHP Version**: PHP 8.2
- **PHP Loader**: ionCube Loader for PHP 8.2

## Required PHP Extensions

The following PHP extensions must be enabled:

- php8.2-common
- php8.2-curl
- php8.2-openssl
- php8.2-json
- php8.2-mbstring
- php8.2-zip
- php8.2-xml

## Recommended Environment

- Linux-based server (Ubuntu / Debian recommended)
- HTTPS enabled (SSL certificate required)
- Memory limit ≥ 256MB
- Allow outbound HTTPS requests (port 443)

> ⚠️ **Note:**  
> ADIYH requires a clean and standard PHP environment.  
> Modified, restricted, or outdated server configurations may cause activation or runtime issues.
