## English

# ADIYH Plugin Installation Guide

This guide explains how to install and activate the **ADIYH WordPress Plugin** correctly.

---

## Installation Steps

### 1. Download the Plugin

Download the **latest version of the ADIYH plugin** from the official member dashboard:

https://adiyh.us/dashboard/downloads

Make sure the file is in **.zip format** and has not been modified.

---

### 2. Upload Plugin to WordPress

1. Log in to your **WordPress Admin (wp-admin)**
2. Go to **Plugins → Add New → Upload Plugin**
3. Upload the **ADIYH .zip** file you downloaded

---

### 3. Activate the Plugin

After the upload is complete, click **Activate Plugin**.

---

### 4. Open ADIYH Menu

After activation:

- Go to the **WordPress sidebar**
- Click **ADIYH**, located just below the **Dashboard** menu

---

### 5. Activate ADIYH Plugin

- The system will automatically ask you to **Activate Plugin**
- Enter the **email registered in your ADIYH member account**

Use the same email as registered on:  
https://adiyh.us/dashboard

If the email is valid and the domain quota is available, the plugin will be activated successfully.

---

## Troubleshooting

### Error: Domain limit exceeded

**Cause:**  
Your domain quota has been exceeded.

**Solution:**

- Remove one domain from your member dashboard  
  or
- Add more domain quota via: Dashboard → Domains → Add Domain Quota

---

### Error: Signature not match

**Cause:**

- Wrong plugin version
- Corrupted plugin file
- Plugin file has been modified

**Solution:**

- Re-download the plugin from:
  https://adiyh.us/dashboard/downloads
- Make sure the plugin file is original and unedited

---

## Notes

- Always use the latest plugin version
- One domain requires one domain quota
- Do not modify plugin files manually

## Indonesian

# Panduan Instalasi Plugin ADIYH

Panduan ini menjelaskan cara menginstal dan mengaktifkan **plugin WordPress ADIYH** dengan benar.

---

## Langkah Instalasi

### 1. Download Plugin

Unduh **versi terbaru plugin ADIYH** melalui dashboard member resmi:

https://adiyh.us/dashboard/downloads

Pastikan file berformat **.zip** dan tidak dimodifikasi.

---

### 2. Upload Plugin ke WordPress

1. Masuk ke **WordPress Admin (wp-admin)**
2. Buka menu **Plugins → Add New → Upload Plugin**
3. Upload file **.zip ADIYH** yang sudah diunduh

---

### 3. Aktifkan Plugin

Setelah upload selesai, klik **Activate Plugin**.

---

### 4. Buka Menu ADIYH

Setelah plugin aktif:

- Pergi ke **sidebar WordPress**
- Klik menu **ADIYH** tepat di bawah **Dashboard**

---

### 5. Aktivasi Plugin ADIYH

- Sistem akan otomatis meminta **Activate Plugin**
- Masukkan **email yang terdaftar di akun member ADIYH**

Gunakan email yang sama dengan yang terdaftar di:  
https://adiyh.us/dashboard

Jika email valid dan domain masih tersedia, plugin akan aktif.

---

## Troubleshooting

### Error: Domain limit exceeded

**Penyebab:**  
Kuota domain Anda sudah habis.

**Solusi:**

- Hapus salah satu domain di dashboard member  
  atau
- Tambahkan kuota domain melalui: Dashboard → Domains → Add Domain Quota

---

### Error: Signature not match

**Penyebab:**

- Salah download file plugin
- File plugin rusak
- File plugin sudah diedit

**Solusi:**

- Download ulang plugin dari:
  https://adiyh.us/dashboard/downloads
- Pastikan file plugin asli dan tidak dimodifikasi

---

## Catatan

- Selalu gunakan versi plugin terbaru
- 1 domain membutuhkan 1 domain quota
- Jangan mengedit file plugin secara manual
