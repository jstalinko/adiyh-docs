## English

## Import Campaign Guide

The Import Campaign feature allows you to duplicate or restore campaign settings without having to configure everything from scratch. This is useful for quickly setting up similar campaigns.

### Prerequisites: Export Campaign

Before importing, you must have a campaign configuration file (`.json`). To get this:

1. Go to **ADIYH → Campaign**.
2. Find the campaign you want to export in the list.
3. In the **Action** column, click the **Arrow Down Icon** ⬇️ (Download).
4. A `.json` file will be downloaded to your computer.

---

### How to Import a Campaign

1. Go to **ADIYH → Campaign**.
2. Click the **Import Campaign** button (usually located at the top).
3. Upload the `.json` file you previously downloaded.

### Import Settings

During the import process, you can adjust several settings to fit the new campaign context:

1. **Replace Slug**  
   Allows you to change the **Slug** configuration from the original `.json` file. This is important to avoid URL conflicts if the original campaign still exists.

2. **Replace Target URL(s)**  
   Allows you to automatically replace the **Target Posts / URLs** based on the list of available posts on your current WordPress site.

3. **Change Themes**  
   Allows you to select or change the **Themes** during the import process, giving the imported campaign a different look.

## Indonesian

## Panduan Import Campaign

Fitur Import Campaign berfungsi agar Anda tidak perlu melakukan pengaturan ulang dari awal. Ini sangat berguna untuk menduplikasi campaign atau memindahkan pengaturan campaign dengan cepat.

### Prasyarat: Export Campaign

Sebelum melakukan import, Anda harus memiliki file konfigurasi campaign (`.json`). Untuk mendapatkannya:

1. Masuk ke **ADIYH → Campaign**.
2. Cari campaign yang ingin Anda export di dalam daftar.
3. Pada kolom **Action**, klik tombol **Icon Panah Bawah** ⬇️ (Download).
4. File `.json` akan terunduh ke komputer Anda.

---

### Cara Import Campaign

1. Masuk ke **ADIYH → Campaign**.
2. Klik tombol **Import Campaign** (biasanya ada di bagian atas).
3. Upload file `.json` yang sudah Anda download sebelumnya.

### Pengaturan Import

Saat melakukan import, ada beberapa pengaturan yang perlu Anda perhatikan:

1. **Replace Slug**  
   Memungkinkan Anda untuk mengganti konfigurasi **Slug** dari file `.json` aslinya. Ini penting untuk menghindari konflik URL jika campaign asli masih ada.

2. **Replace Target URL(s)**  
   Memungkinkan Anda untuk mengganti **Target URL** secara otomatis berdasarkan daftar postingan yang tersedia di website WordPress Anda saat ini.

3. **Change Themes**  
   Memungkinkan Anda untuk mengganti **Themes** (tampilan) saat proses import berlangsung.
