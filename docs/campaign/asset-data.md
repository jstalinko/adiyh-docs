## English

## Asset Data Guide

Before creating a campaign, you must **fill in and complete Asset Data**.  
Asset Data is used to supply dynamic content required by certain landing pages, such as names, titles, images, videos, and keywords.

---

## How to Access Asset Data

1. Go to **WordPress Dashboard**
2. Navigate to **ADIYH → Campaign**
3. On the right side, click the **Asset Data** button
4. Fill in all required data according to your landing page needs

---

## Asset Data Fields Explanation

### DATA LIST NAME

Used to display **person names** on landing pages that require name data.  
Example: WhatsApp chat landing page.

---

### DATA LIST TITLE

Used to display a **list of titles**.  
Example: Video landing pages that require video titles.

---

### DATA LIST PROFILE PIC / AVATAR URL

Used to display **profile pictures or avatars**.  
Commonly used for chat-style or testimonial landing pages.  
You can **upload images directly using WordPress Media Library**.

---

### DATA LIST VIDEOS URL

Used to store **video URLs** required by video-based landing pages.  
Videos can be **uploaded directly via WordPress Media Library**.

---

### DATA LIST BACKGROUND SCREENSHOT URL

Used to display **background screenshots** on landing pages.  
Can be **uploaded directly via WordPress Media Library**.

---

### DATA LIST THUMBNAIL URL

Used to display **thumbnail images** for videos or previews.  
Supports **direct upload via WordPress Media Library**.

---

### DATA KEYWORDS

This data is active when **"Show Article LP"** is enabled in campaign settings.  
Keywords will be **automatically injected** into the landing page.

---

### DATA LOGO IMAGE

Used to display **logo images** on landing pages.  
Supports **direct upload via WordPress Media Library**.

---

### DATA WHITELIST IP (For Test Mode)

Used to define a list of **whitelisted IP addresses** that can access **Test Mode** campaigns.

This allows you to test campaigns without affecting real traffic.

---

## Indonesian

## Panduan Asset Data

Sebelum membuat campaign, Anda **wajib mengisi dan melengkapi Asset Data**.  
Asset Data digunakan untuk menyediakan konten dinamis yang dibutuhkan oleh landing page, seperti nama, judul, gambar, video, dan keyword.

---

## Cara Mengakses Asset Data

1. Masuk ke **Dashboard WordPress**
2. Buka menu **ADIYH → Campaign**
3. Di sebelah kanan, klik tombol **Asset Data**
4. Lengkapi semua data sesuai kebutuhan landing page

---

## Penjelasan Field Asset Data

### DATA LIST NAME

Berfungsi untuk menampilkan **nama orang** pada landing page yang membutuhkan data nama.  
Contoh: landing page WhatsApp chat.

---

### DATA LIST TITLE

Berfungsi untuk menampilkan **daftar judul**.  
Contoh: landing page video yang membutuhkan judul video.

---

### DATA LIST PROFILE PIC / AVATAR URL

Berfungsi untuk menampilkan **foto profil atau avatar**.  
Biasanya digunakan pada landing page chat atau testimonial.  
File dapat **di-upload langsung melalui WordPress Media Library**.

---

### DATA LIST VIDEOS URL

Berfungsi untuk menyimpan **URL video** yang digunakan pada landing page video.  
Video dapat **di-upload langsung melalui WordPress Media Library**.

---

### DATA LIST BACKGROUND SCREENSHOT URL

Berfungsi untuk menampilkan **background screenshot** pada landing page.  
Mendukung **upload langsung melalui WordPress Media Library**.

---

### DATA LIST THUMBNAIL URL

Berfungsi untuk menampilkan **thumbnail** video atau preview konten.  
Mendukung **upload langsung melalui WordPress Media Library**.

---

### DATA KEYWORDS

Field ini aktif jika Anda mengaktifkan **"Show Article LP"** di pengaturan campaign.  
Keyword akan **di-inject secara otomatis** ke landing page.

---

### DATA LOGO IMAGE

Berfungsi untuk menampilkan **logo** pada landing page.  
Mendukung **upload langsung melalui WordPress Media Library**.

---

### DATA WHITELIST IP (Untuk Test Mode)

Digunakan untuk memasukkan daftar **IP whitelist** yang diperbolehkan mengakses **Test Mode** campaign.

Fitur ini memudahkan pengujian campaign tanpa mempengaruhi traffic asli.
