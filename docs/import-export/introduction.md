## English

## What is Import/Export?

The Import/Export feature, introduced in **ADIYH v3.x**, is a comprehensive tool designated for global backups and data migration. It allows you to backup and restore not just campaigns, but your entire ADIYH ecosystem.

This includes:

- **Campaigns**: All your created campaign settings.
- **Ads**: All ad units and configurations.
- **Shortlinks**: All shortlinks you have generated.

### Global vs. Single Campaign Import

It is important to understand the difference between this feature and the standard "Import Campaign" feature:

- **Global Import/Export (This Feature)**:  
  Accessible via `ADIYH > Import/Export`. It works globally for the plugin. It can backup and restore huge chunks of data (Campaigns, Ads, Shortlinks) all at once. It is not compatible with the single campaign export files.

- **Single Campaign Import**:  
  Accessible via `ADIYH > Campaign > Import Campaign`. It is specifically for restoring a single campaign configuration (the `.json` file downloaded from the Campaign list).

Use this Global Import/Export feature when you want to migrate your entire setup to a new site or create a full safety backup.

## Indonesian

## Apa itu Import/Export?

Fitur Import/Export adalah fitur baru yang diperkenalkan mulai **ADIYH v3.x**, yang dirancang sebagai alat backup dan migrasi data secara menyeluruh (global). Fitur ini memungkinkan Anda untuk mencadangkan dan memulihkan tidak hanya campaign, tetapi seluruh ekosistem ADIYH Anda.

Ini mencakup:

- **Campaigns**: Semua pengaturan campaign yang telah Anda buat.
- **Ads**: Semua unit iklan dan konfigurasinya.
- **Shortlinks**: Semua shortlink yang telah Anda buat.

### Perbedaan Global Import vs Single Campaign Import

Penting untuk memahami perbedaan antara fitur ini dengan fitur "Import Campaign" biasa:

- **Global Import/Export (Fitur Ini)**:  
  Diakses melalui `ADIYH > Import/Export`. Ini bekerja secara global untuk plugin. Fitur ini dapat mem-backup dan me-restore data dalam jumlah besar (Campaign, Ads, Shortlink) sekaligus. File dari sini tidak kompatibel dengan fitur import campaign satuan.

- **Single Campaign Import**:  
  Diakses melalui `ADIYH > Campaign > Import Campaign`. Ini khusus digunakan untuk me-restore konfigurasi satu campaign saja (file `.json` yang didapat dari download di daftar campaign).

Gunakan fitur Global Import/Export ini jika Anda ingin memindahkan seluruh pengaturan ke website baru atau membuat backup keamanan yang lengkap.
