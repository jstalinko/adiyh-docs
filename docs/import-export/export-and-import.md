## English

## How to Export & Import Data

This guide covers the technical steps to use the global Import/Export feature in ADIYH.

To access this menu, navigate to:  
**ADIYH → Import/Export**

---

### 1. Export Data (Backup)

Use this feature to create a backup file of your ADIYH data.

1. Go to the **Export** tab or section.
2. Select the data types you want to backup:
   - **Backup Campaigns**: Includes all your campaign configurations.
   - **Backup Ads**: Includes all your ad units.
   - **Backup Shortlinks**: Includes all created shortlinks.
3. Click the **Export** button.
4. A file (usually `.json` or specific ADIYH format) containing all selected data will be downloaded.

---

### 2. Import Data (Restore)

Use this feature to restore data from a Global Export file.

**Note:** This import function is for files generated from the **ADIYH Import/Export** menu only. Do not use single campaign files here.

1. Go to the **Import** tab or section.
2. Upload the file you previously exported from the Global Export menu.
3. Click **Import**.
4. The system will process the file and restore your Campaigns, Ads, and Shortlinks.

## Indonesian

## Cara Export & Import Data

Panduan ini mencakup langkah-langkah teknis untuk menggunakan fitur Import/Export global di ADIYH.

Untuk mengakses menu ini, silakan ke:  
**ADIYH → Import/Export**

---

### 1. Export Data (Backup)

Gunakan fitur ini untuk membuat file cadangan (backup) dari data ADIYH Anda.

1. Masuk ke tab atau bagian **Export**.
2. Pilih tipe data yang ingin Anda backup:
   - **Backup Campaigns**: Termasuk semua konfigurasi campaign Anda.
   - **Backup Ads**: Termasuk semua unit iklan Anda.
   - **Backup Shortlinks**: Termasuk semua shortlink yang telah dibuat.
3. Klik tombol **Export**.
4. File (biasanya format `.json` atau format khusus ADIYH) berisi semua data yang dipilih akan terunduh.

---

### 2. Import Data (Restore)

Gunakan fitur ini untuk memulihkan (restore) data dari file Global Export.

**Catatan:** Fungsi import ini hanya untuk file yang dihasilkan dari menu **ADIYH Import/Export**. Jangan gunakan file backup campaign satuan di sini.

1. Masuk ke tab atau bagian **Import**.
2. Upload file yang sebelumnya Anda export dari menu Global Export.
3. Klik **Import**.
4. Sistem akan memproses file dan memulihkan Campaign, Ads, dan Shortlink Anda.
