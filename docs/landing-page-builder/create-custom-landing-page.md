## English

## How to Create a Custom Landing Page

To create your own landing page using the LP Builder, navigate to:  
**ADIYH → LandingPage Builder**

### The Builder Interface

Once you enter the menu, you will see a dual-pane interface:

- **Left Side**: Raw Code Editor (modify HTML/CSS directly).
- **Right Side**: Live Preview of your code.

### Using AI Assistance

You can speed up the process by using AI.

- Use any external AI you prefer, OR
- Use the **Integrated Gemini AI** by clicking the button in the **bottom right corner**. Input your prompt, and it will generate the code for you.

### Using Blocks / Shortcuts

Can you insert **ADIYH Blocks/Shortcuts** into your code.  
This is essential if you want your page to display dynamic content from your **Asset Data** (like names, images, etc.).

### Saving Your Page

1. Once you are satisfied with your design, look at the top right corner.
2. Enter a **Name** for your landing page in the form field next to the save button.
3. Click **Save**.

## Indonesian

## Cara Membuat Landing Page Custom

Untuk membuat landing page sendiri menggunakan LP Builder, silakan menuju menu:  
**ADIYH → LandingPage Builder**

### Antarmuka Builder

Setelah masuk ke menu tersebut, Anda akan melihat tampilan layar terbagi dua:

- **Sebelah Kiri**: Raw Code Editor (tempat Anda mengedit kode HTML/CSS).
- **Sebelah Kanan**: Live Preview (tampilan hasil kode Anda secara langsung).

### Menggunakan Bantuan AI

Anda bisa mempercepat proses pembuatan dengan AI.

- Gunakan AI eksternal apa saja, ATAU
- Gunakan **Gemini AI yang terintegrasi** dengan menekan tombol di **pojok kanan bawah**. Masukkan perintah (prompt), dan AI akan membuatkan kodenya.

### Menggunakan Blocks / Shortcuts

Anda juga bisa melihat dan memasukkan **Blocks/Shortcuts** ADIYH ke dalam kode Anda.  
Ini penting jika Anda ingin halaman Anda menampilkan data dinamis yang diambil dari **Asset Data** (seperti nama, gambar, dll).

### Menyimpan Halaman

1. Jika desain sudah selesai, perhatikan bagian pojok kanan atas.
2. Masukkan **Nama** untuk landing page Anda pada kolom di sebelah tombol save.
3. Klik **Save**.
