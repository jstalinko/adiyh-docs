## English

## What is Landing Page Builder?

The **Landing Page Builder** in the ADIYH plugin is a powerful tool feature that allows users to create their own custom landing pages effortlessly.

One of its standout capabilities is the deep integration with **Gemini AI**. This allows you to simply input a text prompt, and the AI will generate a landing page layout for you instantly.

### Key Capabilities

1. **Gemini AI Integration**: Create landing pages by just typing a description (prompt) of what you need.
2. **Drag and Drop Blocks**: Easily customize layouts by moving elements around.
3. **ADIYH Asset Data Support**: Fully compatible with ADIYH's built-in shortcuts/blocks (like Data List Name, Images, etc.), ensuring your custom pages can still use dynamic data from the Asset Data menu.

## Indonesian

## Apa itu Landing Page Builder?

**Landing Page Builder** dalam plugin ADIYH adalah fitur canggih yang memungkinkan pengguna untuk membuat landing page sendiri dengan mudah sesuai keinginan.

Salah satu keunggulan utamanya adalah integrasi dengan **Gemini AI**. Hal ini memungkinkan Anda untuk memasukkan perintah teks (prompt), dan AI akan membuatkan struktur landing page secara langsung untuk Anda.

### Kemampuan Utama

1. **Integrasi Gemini AI**: Buat landing page hanya dengan menuliskan deskripsi (prompt) tentang apa yang Anda inginkan.
2. **Drag and Drop Blocks**: Sesuaikan tata letak dengan mudah menggunakan sistem geser dan lepas (drag and drop).
3. **Dukungan Asset Data ADIYH**: Mendukung penuh penggunaan blocks/shortcut bawaan ADIYH (seperti Data List Name, Gambar, dll), memastikan halaman custom Anda tetap bisa menggunakan data dinamis dari menu Asset Data.
