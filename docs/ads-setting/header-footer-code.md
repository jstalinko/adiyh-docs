## English

## Header & Footer Code Settings

This feature allows you to insert custom code into the Header (`<head>`) and Footer (`</body>`) sections of your site or landing pages. This is essential for adding tracking scripts, verification tags, or automated ad codes.

### Usage Guide

1. Go to **ADIYH → Ads Setting**.
2. Navigate to **Header Footer Code** (or scroll to the relevant section).
3. Insert your scripts in the provided fields.

---

### Fields Explanation

#### 1. Header Code

Code placed here will be injected into the `<head>` section.

- **Common Usage**: Google Analytics (`gtag.js`), Google AdSense Auto Ads, verification meta tags, or Facebook Pixel.

#### 2. Footer Code

Code placed here will be injected just before the closing `</body>` tag.

- **Common Usage**: Histats, 3rd party statistic trackers, chat widgets, or other scripts that should load last to avoid blocking page rendering.

#### 3. Display In

Determines where these codes will be active:

- **Plugin & Website**: The code runs on both the ADIYH generated landing pages and your main WordPress site.
- **Plugin Only**: The code runs ONLY on ADIYH landing pages.
- **Website Only**: The code runs ONLY on your main WordPress site.

## Indonesian

## Pengaturan Header & Footer Code

Fitur ini memungkinkan Anda memasukkan kode khusus ke bagian Header (`<head>`) dan Footer (`</body>`) situs atau landing page Anda. Fitur ini sangat penting untuk menambahkan script tracking, verifikasi, atau kode iklan otomatis.

### Panduan Penggunaan

1. Masuk ke **ADIYH → Ads Setting**.
2. Masuk ke bagian **Header Footer Code**.
3. Masukkan script Anda di kolom yang tersedia.

---

### Penjelasan Kolom

#### 1. Header Code

Kode yang dimasukkan di sini akan disisipkan ke dalam tag `<head>`.

- **Kegunaan Umum**: Menempatkan script Google Analytics (`gtag`), Google AdSense Auto Ads, atau meta tag verifikasi.

#### 2. Footer Code

Kode yang dimasukkan di sini akan disisipkan tepat sebelum penutup tag `</body>`.

- **Kegunaan Umum**: Menempatkan script Histats, statistik pihak ketiga (3rd party), atau widget yang sebaiknya dimuat di akhir agar tidak memberatkan loading halaman.

#### 3. Display In

Menentukan di mana kode ini akan aktif:

- **Plugin & Website**: Kode berjalan di landing page ADIYH dan juga di website utama WordPress Anda.
- **Plugin Only**: Kode HANYA berjalan di landing page ADIYH.
- **Website Only**: Kode HANYA berjalan di website utama WordPress Anda.
